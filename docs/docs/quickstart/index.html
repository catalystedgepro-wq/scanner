Find every A-grade DCF — pandas DataFrame in 5 lines

Two-stage Damodaran DCF on 284 international equities. A-grade means ≥100% upside vs current price (sanity-checked, financials capped at 3× to prevent inflation).

PYTHON + PANDAS ~10 sec 
import urllib.request, json
import pandas as pd

d = json.loads(urllib.request.urlopen(
    'https://catalystedgescanner.com/data/intl_dcf.json'
).read())

df = pd.DataFrame(d['top_undervalued'])
a_grade = df[df['grade'] == 'A'][['ticker','name','country','currency','upside_pct']]
print(a_grade.head(10).to_string(index=False))
→10 rows, e.g. ALSEA.MX Alsea Mexico MXN +370.6%

Multi-scanner picks — names firing in 2+ scanners simultaneously

The asymmetric edge layer. When a name appears in cross-border AND DCF AND gap-up, that's institutional positioning.

PYTHON 3 ~5 sec stdlib only 
import urllib.request, json

d = json.loads(urllib.request.urlopen(
    'https://catalystedgescanner.com/data/intl_edge.json'
).read())

for r in d['multi_scanner']:
    if r['scanner_count'] >= 2:
        print(f"{r['ticker']:14s} {r['name'][:25]:25s} ({r.get('country','?'):20s}) → {'+'.join(r['scanners'])}")
→BBCA.JK Bank Central Asia (Indonesia) → dcf_b+gap … RAND.AS Randstad (Netherlands) → dcf_a+gap

Cross-border STRONG setups — Node.js

When the same entity gaps on both home market AND its US ADR with elevated volume on both sides, that's a 4/4 STRONG setup. Bloomberg can't surface this on one screen.

NODE.JS 18+ ~5 sec native fetch 
const r = await fetch('https://catalystedgescanner.com/data/cross_border_convergence.json');
const d = await r.json();

d.top_setups
  .filter(s => s.conviction === 'STRONG' || s.conviction === 'TRADE')
  .forEach(s => {
    console.log(`${s.entity_name}: ${s.foreign_ticker} ${s.foreign_gap_pct.toFixed(2)}% ⇄ ${s.us_ticker} ${s.us_gap_pct.toFixed(2)}% [${s.conviction} ${s.score}/4]`);
  });
→SAP: SAP.DE +4.68% ⇄ SAP +7.36% [TRADE 3/4] · SoftBank: 9984.T +2.16% ⇄ SFTBY +7.26% [TRADE 3/4]
5

Wire to a Slack webhook — one bash command

Get the top multi-scanner pick into your Slack channel. Run on a cron every 2h to mirror our refresh cadence.

BASH + JQ ~10 sec requires Slack incoming webhook 
#!/usr/bin/env bash
WEBHOOK="https://hooks.slack.com/services/YOUR_WEBHOOK_HERE"

PICK=$(curl -s https://catalystedgescanner.com/data/intl_edge.json \
  | jq -r '.multi_scanner[0] | "JACKPOT: \(.ticker) (\(.name)) firing in \(.scanner_count) scanners: \(.scanners | join(\"+\"))"')

curl -X POST -H 'Content-type: application/json' \
  --data "{\"text\":\"$PICK\"}" \
  "$WEBHOOK"
→Slack message: "JACKPOT: BBCA.JK (Bank Central Asia) firing in 2 scanners: dcf_b+gap"

R one-liner — DCF upside distribution

For quants who live in R. Pulls the DCF panel and prints upside-percentage distribution stats.

R + JSONLITE ~5 sec 
library(jsonlite)
d <- fromJSON('https://catalystedgescanner.com/data/intl_dcf.json')
summary(d$top_undervalued$upside_pct)
table(d$top_undervalued$grade)
→Min/Q1/Median/Mean/Q3/Max upside_pct + grade frequency table A=29 B=22 C=28
