v0.1.0 SDK · Python + Node LATEST
Initial public SDK release — Python + Node clients
2026-04-25

First publicly installable SDK. pip install catalyst-edge and npm install @catalyst-edge/sdk. 22 endpoints covered, sync and async clients, CLI bundled, webhook receiver middleware.

  • Python + Node clients — both languages, generated from OpenAPI spec, type-safe end-to-end (Pydantic v2 / TypeScript types)
  • 22 endpoints covered — full surface of the public JSON API: tape, DCF, cross-border, alerts, scanner, international, sympathy
  • Sync + async clients — async iterator for live alerts (async for a in c.alerts.live()); sync clients for one-shot scripts
  • CLI bundled — npx @catalyst-edge/cli scan --grade A prints to stdout; jq-friendly
  • Webhook receiver middleware — Express + FastAPI helpers verify HMAC signatures and decode events
  • Auto-retry with backoff — exponential backoff on 429/5xx; respects Retry-After header
