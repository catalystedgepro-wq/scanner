How the Deep Value Score Works

Each stock is scored 0-100 based on fundamental value metrics. The algorithm weights price-to-book ratio, free cash flow yield, debt levels, insider ownership concentration, return on equity, and short interest pressure.

P/B Ratio — Below book value = underpriced
P/FCF — Price vs free cash flow
Debt/Equity — Lower is safer
Insider Own % — Skin in the game
ROE % — Return on equity
Short Float % — Bearish pressure

Grades: A = Score 50+, B = Score 40-49, C = Score 30-39, D = Below 30. Only stocks with a clear value thesis make the screen.
